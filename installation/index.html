Installing Form Tools is usually very simple. The only problems that usually occur are invalid or missing MySQL credentials, or if you're on a Windows server and the permissions on the theme cache folder aren't right. This section explains how to get the script, upload it to your website, run through the installation steps and deal with the most likely problems. It's really easy! Honest!

Download the script

  1. Download the latest zipfile from the downloads page and unzip the files into a folder on your computer. This file is a package containing various essential modules and themes so you can get the most out of the script, right out of the box. But you can always add and remove modules/themes later.
  2. With an FTP client, upload the files to your web server. The folder you upload them to must be accessible via a URL on your site, e.g. http://www.yoursite.com/formtools/. Unless you want it to be the entire website, you'll want to put it in a subfolder.
  3. After the files have fully uploaded, open your web browser and go to the URL where you uploaded the script. It will automatically take you to the installation script.
