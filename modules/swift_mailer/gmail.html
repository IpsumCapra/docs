Configuration

{% include screenshot.html item="i153.jpg" %}

For those of you with Gmail accounts, you may not know that you can use Gmail's SMTP server to send emails. I use it for all my own Form Tools installations.

  • First you'll need to enable two-factor authentication. This beefs up the security on your Gmail account a little, and is necessary first.
  • Secondly you'll need to create an app password through Google. You will then use this password in lieu of your own gmail password.
