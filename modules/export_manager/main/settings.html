Settings

{% include screenshot.html item="export-manager.png" %}

The Settings page contains a few settings for the module.

  • Export timeout: generating a form data can take time, so this setting let you control the time at which it gives up on the export. By default, it's set to 300 seconds (5 minutes).
  • Generated files folder path / generated files folder URL: when exporting data, the module lets you to generate actual files on the server instead of popping them open in a popup or prompting download through a dialog window. For the file generation option, these settings control which folder should be used to create and store the files.
